Phone as a server

How it came to be

One day I came accross a video on youtube explaining how you can use a phone as a server. That’s correct, servers are just computers that are custom made for a specific need. The processors usually have more cores which run at lower frequencies, they support high speed networking, support for redundant storage and a lot of it, etc. At its core though as long as we’re talking about a small home server a lot of devices could easily pull running lightweight services such as file sharing services, home automation, video streaming and more. This got me to thinking why is it not common sense to reuse old phones this way? Almost everyone has an old one laying around which becomes useless e-waste. Let’s explore the topic together
